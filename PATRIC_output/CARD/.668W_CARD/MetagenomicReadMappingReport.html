  1. Template: Identifier of the template (reference gene) sequence that match the query reads
  2. Function: Template gene function
  3. Genome: Genome that contains template gene
  4. Score: Global alignment score of the template
  5. Expected: Expected alignment score if all mapping reads where smeared over all templates in the database

  7. Template Identity: Percent identity between the query and template sequence, over the length of the matching query sequence
